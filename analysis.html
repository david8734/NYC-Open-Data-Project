This page shows a chart for the leading causes of death in New York City. We use data from the NYC Open Data set and add together all the deaths for each cause. The chart helps us compare different causes using simple bars or slices. When the chart is built, each cause appears with a number that shows how many deaths were reported.

The chart area shows the top causes of death and the total death count for each cause. It puts each cause on the chart using the total number of deaths, so the bigger values are easy to see. That means you can compare causes and see which ones are higher or lower. The chart is built from data rows, and each row adds to the total for one cause. When you look at the chart, you can understand the size of each cause at a glance. This explanation helps make the page easier to understand for someone learning how charts use data. The chart can use different shapes like a bar, a pie, or a donut, but the data is the same. Each shape shows the total death number for each cause in a simple way. This page is meant to teach how raw numbers become a picture on the screen. You can read the explanation first, then use the controls to see the chart appear with the same numbers as the data.
